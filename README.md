this react assignment shows how to organize a React application using multiple components and how Tailwind CSS can be used to quickly build responsive and structured layouts.
It’s a good step towards understanding component reusability and UI organization in modern frontend development.
The Context API in React is used to share data between multiple components without passing props manually at every level. 
It allows you to create a central place (provider) where data is stored, and any component (consumer) can access or update that data easily.
This helps avoid prop drilling and keeps the code cleaner and more manageable, especially in larger applications.That is how the react counter works
